# Sublime Text 2/3 - JavaScript And HTML Snippets

Write code much more faster using some awesome snippets on your Sublime Text.

## How to install

Download
> https://github.com/jrmatos/js-html-snippets.git
> cd js-html-snippets
> mv

## How to use

You just need to write the trigger and hit tab or enter. The trigger is inside the brackets in the list below.

## Snippets

### JavaScript

#### [us] use strict

```js
'use strict';
```

#### [al] alert

```js
alert(obj);
```

#### [cl] console.log

```js
console.log(obj);
```

#### [cw] console.warn

```js
console.warn(obj);
```

#### [ce] console.error

```js
console.error(obj);
```

#### [obj] object literal

```js
{ "attribute": value }
```

#### [tc] try - catch

```js
try {
	// code...
}
catch (e) {
	// code...
}
```

#### for] for

```js
for (var i = 0; i < 10; i++) {
	// code...
}
```
